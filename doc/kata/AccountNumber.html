AccountNumber

class AccountNumber extends AnyRef

Created with IntelliJ IDEA. User: doug Date: 6/6/13 Time: 7:52 PM

This class represents an account number which is made up of nine OcrDigits. This object can either be constructed from a List of nine OcrDigit objects or from a three line, 83 character string formatted number.
